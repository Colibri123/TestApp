Строка "Название фирмы" - ищет в бд данные с похожым название

Строка "Почтовый адрес" - ищет в бд данные с похожым адресом

Строка "Юридический адрес" - ищет в бд данные с похожым адресом

Фильтрация производится следующим образом:

1 -- Вводим название фирмы --
2 -- Вводи почтовый адрес(можно несколько id через ",") --
3 -- Вводим юридический адресс--
4 -- Вывод данных в Grid--
(Можно по отдельности)

Кнопка "Выгрузить данные":

Строит таблицу в шахматном порядке

Поиск данных (кроме названия фирмы) производится по уникальным ID в формате uniqueidentifier
